# FitStart Gym Programs — Complete Reference

This document is the authoritative specification for all four FitStart gym programs. It is intended to be handed to any developer implementing the same programs in a different application. All exercise names, sets, reps, rest periods, tips, warmup/cooldown content, and week-by-week phase progressions are taken directly from `frontend/src/data/programs.js`.

## Implementation Notes for Developers

### Data Model

Each program consists of a flat array of `weeks`, each containing an array of `days`. Every day has the following shape:

```
{
  id: string,              // e.g. "day-1", "w1-d1"
  dayNumber: number,       // overall day counter across the program
  dayName: string,         // e.g. "Monday"
  title: string,           // workout display name
  warmup: {
    duration: string,      // e.g. "5 min"
    exercises: [
      { name: string, duration?: string, sets?: number, reps?: number|string }
    ]
  },
  mainWorkout: [
    { name: string, sets: number, reps: number|string, rest: string, tip?: string }
  ],
  cooldown: {
    duration: string,
    exercises: [
      { name: string, duration: string }
    ]
  }
}
```

### Progressive Overload Logic

- The `tip` field on the first exercise of each session carries the overload instruction (e.g. "Add 2.5 kg vs W1") for weeks where progression applies.
- Within each phase, exercise selection changes. Warmup and cooldown exercises remain constant within a phase for the same day type.
- Upper body and compound push/pull days progress by +2.5 kg per week. Leg-dominant days progress by +5 kg per week.

### Day ID Pattern

| Program | ID pattern | Example |
|---|---|---|
| Free Starter | `day-{n}` | `day-1` |
| Starter | `w{week}-d{dayOfWeek}` | `w3-d2` |
| Transformer | `w{week}-d{dayOfWeek}` | `w5-d3` |
| Elite Beginner | `w{week}-d{dayOfWeek}` | `w10-d4` |

### Day Number Formula

| Program | Formula |
|---|---|
| Starter (3x/wk) | `(week - 1) * 3 + dayIndex` |
| Transformer (4x/wk) | `(week - 1) * 4 + dayIndex` |
| Elite Beginner (5x/wk) | `(week - 1) * 5 + dayIndex` |
